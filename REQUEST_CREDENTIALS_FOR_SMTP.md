## Key Points to Include:

1. **Be Clear About Purpose**: Explain why you need access
2. **Offer Alternatives**: Give multiple options (access, screenshot, or manual check)
3. **Security Assurance**: Mention you'll only access what's needed
4. **Time Frame**: Mention it's temporary/one-time
5. **Professional Tone**: Keep it friendly but professional
6. **Make It Easy**: Offer to guide them if they prefer not to share

---

## What to Check Once You Have Access:

1. Go to Hostinger Email Panel
2. Navigate to: Email Accounts → `contact@ebyapar.com`
3. Look for: "SMTP Settings", "Email Client Configuration", or "Outgoing Mail"
4. Note down:
   - SMTP Host
   - SMTP Port
   - Encryption (SSL/TLS)
   - Username format
   - Any special requirements
5. Check if SMTP needs to be enabled (toggle/checkbox)

---

## After Getting Credentials:

1. Verify SMTP settings
2. Update `.env` file with correct settings
3. Test SMTP connection
4. Inform client that access is no longer needed
5. Recommend they change password if they shared it
