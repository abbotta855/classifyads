# Mailtrap Setup Steps

## Step 1: Sign Up / Login
1. Go to https://mailtrap.io
2. Click "Sign Up" (or "Log In" if you already have an account)
3. You can sign up with:
   - Email
   - Google account
   - GitHub account

## Step 2: Create an Inbox
1. After logging in, you'll see your dashboard
2. Click "Add Inbox" or "Create Inbox"
3. Give it a name (e.g., "My App Testing")
4. Click "Create"

## Step 3: Get SMTP Credentials
1. Click on your inbox
2. You'll see different integration options
3. Click on "SMTP Settings" tab
4. Select "Laravel" from the dropdown (or use "Integrations" â†’ "Laravel")
5. You'll see credentials like:

   - **Port:** 2525 (or 587)
   - **Username:** (a long string)
   - **Password:** (a long string)
